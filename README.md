# 시니어 세대를 위한 건강 기록 앱

## 🥈 제 10회 미래 내일 일경험 프로젝트 (은상)
- 고용 노동부에서 진행하는 프로그램으로, 기업이 제안한 프로젝트를 수행하기 위해 수행 과정에 대한 코칭 및 평가를 받아 직무역량을 강화하는 프로그램
- 시니어 세대들의 건강 관리를 위해 도움을 줄 수 있는 프로젝트를 제안 받아 진행한 건강 기록 앱
